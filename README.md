**`Data Analyst`**

Analista de Dados Pleno na Shopee, atuando em Data & Analytics no time de Afiliados. Responsável pela estrutura analítica da área, com foco em segmentação de usuários, análise de performance e construção de dashboards estratégicos, incluindo iniciativas com parceiros como YouTube e Meta.

Iniciei minha trajetória na Rappi, onde evoluí de Jovem Aprendiz a Analista de Dados Júnior em SP&A. Nesse período, desenvolvi dashboards, automatizei processos analíticos e implementei análises geoespaciais para suportar decisões de negócio.

Atualmente, aplico conceitos de estatística, BI e Data Science para gerar insights e apoiar decisões estratégicas na Shopee.
